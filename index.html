bootmaker An automated Bootstrap workflow

An automated Bootstrap workflow in the simplified form of a Makefile.

Download the Makefile from your commandline.

Run the default make command to build Bootstrap.

Automate your workflow with the make watch command.

Check it out on Github → Github

Why should I use this?

You're proficient on the commandline and want a simple and automated way to build Bootstrap sites, preferably using Github Pages. This project aims to simplify your workflow and get out of your way.

How is this helpful?

As of right now, the end goal is an easy way to jumpstart development and help with automation. In the future, more commands will be added to help with upgrading your current version of Bootstrap.

Why should I not use this?

You're uncomfortable using the commandline, Github Pages, or Bootstrap. Also, you might not like small utility scripts and/or nice things.
